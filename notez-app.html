Notez App

Mobile App Development

This was a project done for my Mobile App class. The goal of the project was to create any app you wanted, the only requirement was to use NativeScript code.

I decided to create Notez. A common note-taking platform where students within school can take notes, share with others, comment on notes, add to existing notes or make new notes about a particular topic, lecture or seminar. A simple, sleak, and easily navigational app was my goal for this project.
